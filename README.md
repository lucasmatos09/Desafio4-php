# 📘 Exercícios de Lógica com PHP

Este projeto reúne uma coleção de **15 exercícios de lógica de programação utilizando PHP**, com foco em:

* Manipulação de formulários HTML
* Validação de dados com `$_POST`
* Uso de `filter_var()`
* Estruturas condicionais (`if`, `switch`)
* Laços de repetição (`for`, `foreach`)
* Manipulação de arrays

Tudo foi desenvolvido em **um único arquivo PHP**, com um layout moderno e organizado em formato de **cards interativos**.

---

## 🚀 Funcionalidades

O sistema contém os seguintes exercícios:

### 🔢 Bloco 1 – Cálculos Simples

1. Conversor de Moedas
2. Área e Perímetro de Retângulo
3. Consumo de Combustível

### ⚖️ Bloco 2 – Condicionais (if/else)

4. Média do Aluno
5. Verificação de Votação por Idade
6. Par ou Ímpar

### 🔀 Bloco 3 – Condicionais (switch/case)

7. Dia da Semana
8. Calculadora Simples
9. Mês por Extenso

### 🔁 Bloco 4 – Laços de Repetição

10. Fatorial
11. Somatório de 1 até N
12. Números Pares em Intervalo

### 📊 Bloco 5 – Arrays

13. Média de Vários Valores
14. Lista de Compras (Checkbox)
15. Maior Número Digitado
